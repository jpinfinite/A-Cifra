**[Filecoin](/artigo/filecoin-fil-almacenamiento-descentralizado-de-datos "Filecoin (FIL): Almacenamiento Descentralizado de Datos")** is a decentralized storage network that allows anyone to rent storage space.

## Centralized Storage Problem

- **Censorship**: Centralized control
- **Cost**: High prices
- **Privacy**: Exposed data
- **Availability**: Single points of failure

## How Filecoin Solves It

### Proof of Replication

Cryptographic proof that data is stored.

### Proof of Spacetime

Continuous proof of storage over time.

## Token FIL

- **Payment**: For storage
- **Staking**: Storage providers
- **Collateral**: Service guarantee

## IPFS vs Filecoin

- **IPFS**: Storage protocol
- **Filecoin**: Incentivization layer

## Use Cases

- **NFTs**: Permanent storage
- **Files**: Decentralized backup
- **Video**: Decentralized streaming
- **Data**: Historical preservation

## Conclusion

[Filecoin](/artigo/filecoin-armazenamento-descentralizado "Filecoin (FIL): Armazenamento Descentralizado de Dados") is building the storage infrastructure for Web3.
