Desde a sua criação, a tecnologia [blockchain](/artigo/avalanche-subnets-and-blockchain-scalability "Avalanche: Subnets and Blockchain Scalability") tem sido capaz de revolucionar a forma como as transações são realizadas e os contratos são executados. A ideia de um registro imutável e descentralizado de dados foi um passo gigante em direção a uma sociedade mais justa e transparente. No entanto, a introdução da tecnologia blockchain também levantou uma série de desafios. Um dos principais problemas é a falta de conectividade entre o mundo virtual e o mundo real. Enquanto os blockchains são ambientes isolados, não podem acessar dados externos diretamente, o que torna difícil para os contratos inteligentes executarem ações baseadas em informações do mundo real.

Essa foi a razão pela qual os oráculos foram criados. Eles são serviços que fornecem dados externos para os contratos inteligentes, permitindo que eles sejam executados de forma mais eficaz. No entanto, os oráculos também têm seus próprios problemas. A maioria deles é centralizada, o que significa que um único ponto de falha pode levar a uma perda de confiança geral. Além disso, os oráculos podem ser manipulados ou censurados, o que pode levar a resultados imprevisíveis.

Com uma rede de oráculos descentralizados, o Chainlink permite que os contratos inteligentes sejam executados de forma mais segura e confiável. Isso é possível porque a rede é descentralizada, o que significa que não há um único ponto de falha. Além disso, a rede é transparente e imutável, o que garante que os dados sejam precisos e confiáveis.

## Fundamentos Técnicos de Chainlink

### Introdução

Chainlink é uma plataforma de oráculos descentralizados que conecta smart contracts a dados do mundo real de forma segura e confiável. Para entender como Chainlink funciona, é necessário conhecer os fundamentos técnicos por trás da plataforma.

### O Problema da Conectividade

Como mencionado anteriormente, blockchains são ambientes isolados que não podem acessar dados externos diretamente. Isso é semelhante a um computador que não pode acessar a internet sem uma conexão de rede. Para resolver esse problema, os smart contracts precisam de uma forma de acessar dados externos, que é onde os oráculos entram em cena.

### Como os Oráculos Funcionam

Os oráculos são serviços que fornecem dados externos para os smart contracts. Eles podem ser comparados a uma biblioteca que fornece informações sobre um determinado assunto. No entanto, ao invés de armazenar informações em um livro, os oráculos armazenam informações em uma rede descentralizada.

#### Tipos de Oráculos

Existem dois tipos de oráculos em Chainlink:

* **Oráculos de Dados**: fornecem dados específicos, como preços de ações ou taxas de câmbio.
* **Oráculos de Funções**: fornecem resultados de funções matemáticas, como cálculos de juros compostos.

#### Como os Oráculos são Conectados

Os oráculos são conectados à rede de Chainlink por meio de uma rede de nós. Cada nó é um computador que executa o software do Chainlink e fornece dados para os smart contracts. Os nós são distribuídos em todo o mundo e são responsáveis por fornecer dados para os smart contracts.

#### Como os Dados são Verificados

Os dados fornecidos pelos oráculos são verificados por meio de uma combinação de técnicas de verificação, incluindo:

* **Hashes**: os dados são criptografados e armazenados em uma forma que não pode ser alterada.
* **Certificados de Verificação**: os nós que fornecem dados são identificados e verificados por meio de certificados de verificação.
* **Consensus**: os dados são verificados por meio de um processo de consenso, em que os nós da rede concordam sobre a validade dos dados.

### Conclusão

Chainlink é uma plataforma de oráculos descentralizados que conecta smart contracts a dados do mundo real de forma segura e confiável. A plataforma utiliza uma rede de nós distribuídos em todo o mundo para fornecer dados para os smart contracts, e utiliza técnicas de verificação para garantir a integridade dos dados. Com Chainlink, os desenvolvedores podem criar smart contracts que sejam mais robustos e confiáveis, e que possam acessar dados externos de forma segura e eficiente.
