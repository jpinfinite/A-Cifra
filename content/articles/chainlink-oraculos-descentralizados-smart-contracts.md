**Chainlink** é a infraestrutura de oráculos descentralizados líder de mercado, conectando smart contracts a dados do mundo real, APIs e sistemas de pagamento tradicionais.

## O Problema dos Oráculos

### Limitação dos Smart Contracts

Smart contracts são programas auto-executáveis na blockchain, mas têm uma limitação fundamental: não podem acessar dados externos por conta própria. Eles precisam de oráculos para trazer informações do mundo real.

### O Problema do Oráculo

- **Centralização**: Oráculo único = ponto de falha
- **Manipulação**: Dados podem ser alterados
- **Confiabilidade**: Como garantir precisão?
- **Disponibilidade**: E se o oráculo ficar offline?

## Como Chainlink Resolve

### Rede Descentralizada

Chainlink usa múltiplos nós independentes para fornecer dados, eliminando pontos únicos de falha e aumentando a confiabilidade.

### Agregação de Dados

- Múltiplas fontes de dados
- Consenso entre nós
- Detecção de outliers
- Penalização de nós maliciosos

## Casos de Uso

### DeFi

- **Price Feeds**: Preços de ativos em tempo real
- **Proof of Reserves**: Verificação de colateral
- **Liquidações**: Triggers automáticos
- **Yield Farming**: Cálculo de APY

### Seguros

- **Seguros Paramétricos**: Pagamentos automáticos
- **Dados Climáticos**: Condições meteorológicas
- **Voos**: Atrasos e cancelamentos

### Gaming e NFTs

- **Randomness**: Números aleatórios verificáveis
- **Dynamic NFTs**: NFTs que mudam com dados reais
- **Torneios**: Resultados esportivos

## Token LINK

### Utilidade

- **Pagamento**: Nós recebem LINK por dados
- **Staking**: Garantia de bom comportamento
- **Governança**: Futuras decisões de protocolo

### Tokenomics

- **Staking Rewards**: 5-7% APY (estimado)

## Chainlink 2.0

### Novos Recursos

- **Staking**: Segurança econômica

## Conclusão

Chainlink é essencial para o futuro dos smart contracts, conectando blockchain ao mundo real de forma segura e descentralizada.
