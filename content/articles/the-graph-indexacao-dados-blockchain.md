# The Graph (GRT): Indexação de Dados Blockchain

The Graph é o protocolo de indexação para consultar dados de blockchains como Ethereum e IPFS, funcionando como o "Google das blockchains".

## O Problema da Consulta de Dados

Consultar dados diretamente de blockchains é extremamente ineficiente:

### Desafios
- **Lento**: Varrer toda a blockchain leva tempo
- **Caro**: Muitas chamadas RPC custam caro
- **Complexo**: Dados não estruturados
- **Ineficiente**: Sem indexação adequada

### Impacto
- dApps lentos
- Experiência ruim do usuário
- Custos elevados de infraestrutura

## A Solução: The Graph

The Graph resolve isso com um protocolo de indexação descentralizado:

### Como Funciona
1. **Subgraphs**: Definem quais dados indexar
2. **Indexadores**: Processam e indexam dados
3. **Curadores**: Sinalizam subgraphs de qualidade
4. **Delegadores**: Fazem stake em indexadores
5. **Consumidores**: Consultam dados pagando em GRT

## Componentes do Protocolo

### Subgraphs
- APIs abertas para dados blockchain
- Definem esquema de dados
- Especificam eventos a indexar
- Podem ser compostos

### Indexadores
- Operam nós que indexam dados
- Fazem stake de GRT
- Ganham taxas de consulta
- Penalizados por má conduta

### Curadores
- Sinalizam subgraphs valiosos
- Fazem stake de GRT em subgraphs
- Ganham parte das taxas
- Ajudam a descobrir qualidade

### Delegadores
- Delegam GRT para indexadores
- Ganham recompensas
- Não precisam rodar nós
- Compartilham riscos e recompensas

## O Token GRT

GRT é o token de utilidade do protocolo:

### Funções
- **Pagamento**: Taxas de consulta
- **Staking**: Indexadores e curadores
- **Delegação**: Delegadores ganham recompensas
- **Governança**: Decisões do protocolo

### Tokenomics
- Inflação para recompensas
- Queima de taxas de consulta
- Slashing por má conduta

3. Deploy no Graph Network

- Sem configuração necessária

## Staking e Delegação

### Como Fazer Staking
**Indexadores**
- Stake mínimo: 100.000 GRT
- Rodar infraestrutura
- Ganhar taxas de consulta

**Curadores**
- Sinalizar subgraphs
- Stake em bonding curves
- Ganhar parte das taxas

**Delegadores**
- Delegar para indexadores
- Sem mínimo
- Recompensas de 10-15%

### Riscos
- Slashing por má conduta
- Volatilidade do GRT
- Riscos de smart contracts

## The Graph vs Alternativas

### The Graph vs Serviços Centralizados
- **Descentralização**: Sem ponto único de falha
- **Censura**: Resistente
- **Transparência**: Código aberto

### The Graph vs Indexação Própria
- **Custo**: Mais barato
- **Manutenção**: Sem overhead
- **Escalabilidade**: Automática

The Graph é infraestrutura essencial para Web3, permitindo que dApps consultem dados blockchain de forma eficiente e descentralizada.
