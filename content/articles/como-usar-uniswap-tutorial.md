O **Uniswap** é a maior DEX (exchange descentralizada) do mundo. Este tutorial ensina a usar a plataforma passo a passo.

## O Que Você Precisa

- Carteira MetaMask instalada
- ETH para taxas de gas
- Tokens para trocar

## Passo 1: Acesse o Uniswap

1. Vá para app.uniswap.org
2. Clique em "Connect Wallet"
3. Selecione MetaMask
4. Aprove a conexão

## Passo 2: Fazer um Swap

1. Selecione o token que tem (ex: ETH)
2. Selecione o token que quer (ex: USDC)
3. Digite a quantidade
4. Verifique a taxa e slippage
5. Clique em "Swap"
6. Confirme na MetaMask

## Entendendo Slippage

- Diferença entre preço esperado e executado
- Recomendado: 0.5-1%
- Tokens menos líquidos: até 3%

## Taxas de Gas

- Variam conforme congestionamento
- Use gwei tracker para economizar
- Evite horários de pico

## Dicas de Segurança

✅ Sempre verifique o endereço do token
✅ Use slippage adequado
✅ Comece com valores pequenos
❌ Nunca compartilhe sua seed phrase
