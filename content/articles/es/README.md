# Artículos en Español

Esta carpeta contiene los artículos traducidos al español.

## Estructura del Archivo

Los archivos deben ser Markdown (`.md`) con el siguiente frontmatter:

```yaml
---
id: 'bitcoin-analisis-2025'
title: 'Bitcoin: Análisis Completo para 2025'
slug: 'bitcoin-analisis-2025'
excerpt: 'Resumen del artículo...'
coverImage:
  src: '/images/cover.jpg'
  alt: 'Bitcoin Cover'
author:
  name: 'Jonatha Pereira'
publishedAt: '2025-12-08'
categorySlug: 'bitcoin'
tags: ['bitcoin', 'criptomonedas', 'análisis']
language: 'es'
alternateLanguages:
  pt: 'bitcoin-analise-2025'
  en: 'bitcoin-analysis-2025'
seo:
  metaTitle: 'Bitcoin 2025: Análisis Completo'
  metaDescription: 'Descripción para SEO...'
  keywords: ['bitcoin', 'btc', 'crypto']
---

Contenido del artículo en español...
```

## Flujo de Trabajo

1. Crear un archivo .md en esta carpeta.
2. Asegurar que el slug sea único.
3. Vincular con versiones en PT y EN usando `alternateLanguages`.
