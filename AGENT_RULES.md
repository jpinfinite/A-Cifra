# Antigravity Rules – A Cifra

## Conteúdo
- Todo artigo novo deve ser criado em src/content/articles
- Nunca criar conteúdo dentro de components ou app

## Imagens
- Todas as imagens vão para public/images/articles
- Formato obrigatório: .webp
- Nome do arquivo = slug do artigo
- Nunca usar domínio absoluto para imagens

## Código
- Não criar novos loaders de imagem
- Reutilizar componentes existentes
- Não duplicar lógica de SEO
- Não alterar next.config.js sem justificativa

## SEO
- Sempre gerar title, description e alt de imagem
- Sempre gerar schema Article
