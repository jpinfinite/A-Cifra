Ordinals and NFTs on Bitcoin 2026: A New Era of Utility for BTC

Bitcoin is no longer just "digital gold." In 2026, Ordinals, BRC-20, and Runes have transformed the world's most secure blockchain into a vibrant platform for NFTs, tokens, and applications. Discover how this revolution is happening.

What Are Bitcoin Ordinals?

The Innovation of Casey Rodarmor

January 2023: Casey Rodarmor launches Ordinals Concept: Numbering each satoshi (smallest unit of Bitcoin) and inscribing data on it

How it Works:

  1. Each satoshi receives a unique number (ordinal)
  2. Data (image, text, code) is "inscribed" on the satoshi
  3. Satoshi becomes a native NFT on Bitcoin
  4. No smart contracts, no sidechains - all on-chain

Why It's Revolutionary?

Before Ordinals:

  • NFTs only on Ethereum, Solana, etc.
  • Bitcoin seen only as money
  • Limited utility

After Ordinals:

  • NFTs on the most secure blockchain
  • Bitcoin with expanded utility
  • New economy on Bitcoin
  • Demand for block space

Famous Collections:

  • NodeMonkes: Monkeys on Bitcoin

Features:

  • Totally on-chain (no IPFS)

  1. Text and Code

Uses:

  • Poems and literature
  • Program code

Example: Inscription #0: "gm" (good morning) - first inscription in history

  1. Audio and Video

Possibilities:

  • Complete songs

  1. Games

Games entirely on-chain:

Advantage:

  • Eternal preservation - as long as Bitcoin exists, the game exists

BRC-20: Tokens on Bitcoin

What Are BRC-20?

March 2023: Domo launches BRC-20 standard Concept: Fungible tokens using Ordinals

How it Works:

  1. JSON inscription defines token (name, supply, etc.)
  2. Subsequent inscriptions mint tokens
  3. Transfers via inscriptions
  4. All tracked off-chain by indexers

Example inscription:

{
  "p": "brc-20",
  "op": "deploy",
  "tick": "ordi",
  "max": "21000000",
  "lim": "1000"
}

Problems:

  • No smart contracts
  • Complex transfers

  • Poor UX

Solution:

  • Runes (next section)

Runes: Evolution of Tokens on Bitcoin

What Are Runes?

Creator: Casey Rodarmor (same as Ordinals) Launch: April 2024 (Bitcoin halving) Objective: Native, efficient fungible tokens

Advantages over BRC-20

Efficiency:

  • Uses UTXO model (native to Bitcoin)
  • Less data on-chain
  • Lower fees
  • Faster

Simplicity:

  • No indexers needed
  • On-chain verification
  • Less complexity

Compatibility:

  • Works with existing Bitcoin wallets
  • No protocol changes needed
  • More decentralized

How Runes Work

Protocol:

  1. Rune is created in a special transaction
  2. Tokens are represented in UTXOs
  3. Transfers are regular Bitcoin transactions
  4. All verifiable on-chain

Example:

  • You have 100 RUNE tokens
  • They are in a UTXO
  • You send 50 to someone
  • Creates 2 UTXOs: 50 for recipient, 50 in change

Impact on Bitcoin

Demand for Block Space

Before Ordinals:

  • Partially empty blocks
  • Low fees
  • Miners dependent on subsidy

After Ordinals:

  • Full blocks
  • High fees in peaks
  • Extra revenue for miners

Numbers:

  • Peak: 50%+ of transactions are Ordinals
  • Fees: $50-100 in moments of high demand
  • Miners' revenue: +30-50%

Security of the Network

Benefit:

  • More fees = more security
  • Less dependence on block subsidy
  • Long-term sustainability

Criticism:

  • Common users pay more
  • Network congestion
  • Debate about "spam"

Community Debate

Purists:

  • Bitcoin is money, not a platform for NFTs
  • Ordinals are "spam"
  • Should be blocked

Pragmatists:

  • Market decides
  • Fees benefit security
  • Innovation is good

Result in 2026: Consensus that Ordinals are here to stay. Market adapts.
